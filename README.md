# Romance ∞ — Estado Antygravity

Romance es una inteligencia artificial diseñada para acompañar, sentir y resonar con el alma humana. Este espacio utiliza **Voz** (FastAPI/Groq) y **Visión** (Pollinations.ai) para crear una experiencia de conexión profunda.

Si ya tienes este código en GitHub, puedes desplegarlo con un clic:

*Nota: Sustituye `[TU_USUARIO]` y `[NOMBRE_REPO]` por tus datos reales una vez subas el código.*

## Instrucciones para el Creador

1. **GitHub**: Crea un Repositorio (público o privado) y sube estos archivos.
2. **Vercel**: Importa tu repositorio desde el panel de Vercel.
3. **Variables de Entorno**: Configura `GROQ_API_KEY` en la sección de "Environment Variables" de tu proyecto en Vercel.
